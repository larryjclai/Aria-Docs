AMA = ask me anything 為現在時下流行的用語。

《數位廣告AMA》為非營利單位，主要是希望用通俗易懂的口語，來讓大家了解數位廣告到底在幹嘛、遇到問題了該怎麼處理。

我們會嘗試將主題分成 Meta 廣告、Google 廣告以及 LINE 廣告幾個大家常見的廣告平台，讓你能夠針對你的問題更快地找到答案。

如果翻不到你想要的內容，可以嘗試用右上角的搜尋功能來做搜尋哦！或是可以提交相關問題給我們，我們會在作出回覆。
